## 📘 Java 학습
### ▸ 자바의 정석 예제 실습
* **3판(Java 8 기준)**: 기본 문법, 객체지향, 컬렉션, 제네릭, 예외 등
  ⤷ `java-basic-3rd/` 디렉토리에 정리
* **4판(Java 21 기준)**: 람다식, 스트림, record, switch 확장 등 최신 기능 포함
  ⤷ `java-basic-4th/` 디렉토리에 정리
📌 실습 중심으로 **출제 예제 + 자체 변형 문제 풀이** 포함
