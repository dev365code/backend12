# 축구는 종교다. Soccer Is Religion.
축구, 그것은 스포츠 이상.

## 프로젝트 소개 Introduction
스프링을 통한 쇼핑몰 기본&심화 기능 구현이 목적이며, 세부 기능은 아래와 같음
1. 회원가입 & 로그인 기능
2. 상품 목록 & 상세 내용 기능
3. 주문 & 장바구니 기능
4. 게시판 기능
